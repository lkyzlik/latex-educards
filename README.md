# EduCards

LaTeX macros to create information cards with colored borders.

## Usage
'''
\usepackage{educrads}
'''
'''
\begin{educardf}{color}{category_icon}{category_name}{title}{subtitle}{image_path}
Card front content
\end{educardf}

\begin{educardb}{color}{category_icon}{category_name}{title}
Card back content
\end{educardb}
'''
